How To Play Jacob's Minesweeper

Two player does not currently work :( - it is my goal

  1. Choose Difficulty to begin - if not familiar with minesweeper I suggest the default - Easy
  2. Click on 1 player start to start the game
  3. Click any random box to reveal if it is a mine or is a box near a mine
  4. The number inside the box tells you how many mines are within a one block space of that box - above/below, left/right, and diagonally
  5. When a box is not near a mine it will reveal other nearby boxes not near mines
  6. If you think a hidden box has a mine - Right Click or ctrl-left Click to flag it
  7. To win the game you must flag all of the mines without revealing them
  8. left clicking on a mine is an Automatic loss
  9. Press Reset to start over - must click Player Start to begin new game
